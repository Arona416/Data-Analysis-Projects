# Shipping Data Cleaning using SQL

## Project Overview

This project demonstrates a complete SQL data cleaning workflow using Google BigQuery.

The dataset contains shipment records with inconsistent values, multiple date formats, missing values and duplicate detection.

---

## Dataset

- Shipping Dataset
- 25 Records
- 11 Features

---

## Data Cleaning Steps

- Explore the dataset
- Standardize shipment status
- Standardize carrier names
- Standardize warehouse names
- Standardize damage reported values
- Standardize date formats
- Detect duplicate records
- Check missing values
- Create cleaned dataset
